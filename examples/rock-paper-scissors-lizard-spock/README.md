## Run

With the **GruPHst** repo cloned on your machine:

`$ cargo run`

Or just with this example on your computer:

On *Cargo.toml* delete the line with:

`gruphst = { path = "./../../../gruphst" }` 

then execute:

`$ cargo add gruphst && cargo run`
